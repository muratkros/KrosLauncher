KrosLauncher v1.0
---------------------------------------------------------------------------------
KrosLauncher, oyuncular için geliştirilmiş, yüksek performanslı ve modern arayüzlü bir oyun başlatıcıdır (Launcher). Standart başlatıcıların ağırlığından kurtulup, sadece en sevdiğiniz oyunlara en hızlı şekilde ulaşmanızı sağlar.
Özellikler:
Performans:GPU veya CPU yormayan,düşük ram kullanımı.

Zaman Takibi: Her oyunun toplam oynama süresini otomatik olarak kaydeder.

JSON Tabanlı Veri Yönetimi: Oyun verilerinizi yerel diskte güvenli ve hızlı bir şekilde saklar.

Yüksek Performans: C++ ve Qt 6 altyapısı sayesinde minimum RAM kullanımı ve anlık tepki süresi.
----------------------------------------------------------------------------------
Key Features
Optimized Performance: Minimal GPU and CPU load with very low RAM consumption.

Playtime Tracking: Automatically logs and tracks the total playtime for every game in your library.

JSON-Based Data Management: Stores all game data securely and lightning-fast on your local disk.

High Efficiency: Powered by C++ and Qt 6 infrastructure for a minimum memory footprint and instant responsiveness.
----------------------------------------------------------------------------------
Nasıl Kullanılır?
Release kısmından en güncel .zip dosyasını indirin.

Klasöre çıkartın ve appKrosLauncher.exe dosyasını çalıştırın.

+ OYUN EKLE butonuna basarak önce oyunun .exe dosyasını, ardından kapak resmini seçin.

PLAY butonuna basarak uçuşa geçin!

----------------------------------------------------------------------------------
How to use?
Download the latest .zip package from the Releases section.

Extract the archive and launch appKrosLauncher.exe.

Click the + ADD GAME button.

Step 1: Select the game's .exe file.

Step 2: Select your custom cover image (.jpg/.png).

Hit the PLAY button and enjoy the ride!
---------------------------------------------------------------------------------
Tech.
Language: C++

Framework: Qt 6.11.0

UI: QML & Qt Quick Controls 2
